# GEMM v1

An optimized version of [GEMM v0](../02-gemm-v0/README.md).
Benchmarks are added and your solution must pass them.
The formulation is simplified: only $RowMajor \times ColMajor = ColMajor$ matrix multiplication needs to be supported.
